# Kalibra

Self-Evaluating Visual Inspection

Kalibra is an AI engineering project exploring trustworthy visual inspection for industrial quality control.

Rather than only detecting visual anomalies, Kalibra is designed to evaluate when its own predictions can be trusted through calibration, uncertainty estimation, drift awareness, and human-review routing.
